# elasticsearch set #


---
## 各个压缩包说明
* elasticsearch.bz2 - es模板
* plugin_script.bz2 - 包含了load.sh & plugins & kibana
* es_stat.bz2 - 统计脚本


---
## 集群生成
* 创建一个文件夹
* 把elasticsearch放入
* 把plugin_script/load.sh放入
* 把plugin_script/plugins 放入
* 按照load.sh的命令提示创建并启动即可
