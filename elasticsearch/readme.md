# elasticsearch set #


---
## 各个压缩包说明
* es.bz2 - es模板  
  es的版本是2.4.6，另外有load.sh和plugins(analysis-kuromoji, bigdesk, head, kopf, license, marvel-agent, sql)
* es_stat.bz2 - 统计脚本

---
## 集群生成
* 创建一个文件夹
* 把elasticsearch放入
* 按照load.sh的命令提示创建并启动即可
* es版本是2.4.6，对应的kibana版本是4.6.5，去https://www.elastic.co/downloads/past-releases下载
